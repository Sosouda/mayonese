Инструкция по использованию калькулятора MK45

Следуйте приведенным ниже инструкциям для правильного использования калькулятора MK45.

Основные клавиши:

  • 0...9 - клавиши ввода цифр
  • . - клавиша ввода запятой
  • +, -, Х, ÷ - клавиши для выполнения операций сложения, вычитания, умножения и деления
  • /-/ - клавиша изменения знака числа или порядка
  • [( , )] - клавиши для работы с двухуровневыми скобками
  • ВП - клавиша ввода порядка чисел
  • π - клавиша вызова числа пи
  • С - клавиша общего сброса. Для очистки МК необходимо двухкратное нажатие клавиши С.
  • √ - клавиша извлечения квадратного корня
  • 1/x - клавиша вычисления обратной величины числа
  • n! - клавиша вычисления факториала
  • у^х - клавиша возведения числа у в степень х
  • sin, cos, tg - клавиши для вычисления тригонометрических функций
  • arc - клавиша для вычисления обратных тригонометрических функций
  • ln, lg, 10^x, e^x - клавиши для вычисления логарифмов
  • Г->Р, Р->Г - клавиши для перевода величин между градусами и радианами

Примечание: При нажатии клавиши С не происходит сброс регистра памяти.
